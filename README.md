# 🍳 Chef Claude — AI Recipe Generator

Chef Claude is an AI-powered recipe generator app built with **React.js** and **Google Gemini API**.  
It helps users create delicious and creative recipes based on available ingredients, cuisines, and dietary preferences.

---

## ✨ Features
- 🥗 **Ingredient-based recipe generation** — enter ingredients you have, and Chef Claude will suggest recipes.
- 🌍 **Cuisine selection** — get recipes inspired by different cuisines.
- ⚡ **Powered by Google Gemini AI** — generates unique, step-by-step cooking instructions.
- 🎨 **Modern UI** — clean, responsive design built with React.
- 💾 **Save favorite recipes** (planned feature).
- 🔊 **Voice assistant support** (future enhancement).

---

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite (or CRA if you used it)
- **AI Model:** Google Gemini API
- **Styling:** Tailwind CSS (or your chosen framework)
- **Version Control:** Git + GitHub

---

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Santhanakrishnan2004/chefclaude.git
cd chefclaude
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Environment Variables

Create a `.env` file in the root directory and add your Google Gemini API key:

```bash
VITE_GEMINI_API_KEY=your_api_key_here
```

> ⚠️ Never commit your `.env` file. Keep your API keys safe.

### 4. Run the App

```bash
npm run dev
```

Then open: [http://localhost:5173](http://localhost:5173)

---

## 📦 Build for Production

```bash
npm run build
```

Output will be in the `/dist` folder.
